# Contributing

Thanks for improving Vagus Reset Coach.

## Local Workflow

```bash
npm install
make install-hooks
make lint
make test
make build
make smoke
```

Use Conventional Commits for every commit, for example `feat: add pacer session summary`.

## Privacy And Safety

Do not add network uploads for webcam frames, biometrics, or session history. V1 is intentionally local-first and GitHub Pages only.
